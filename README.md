# World Explorer

World Explorer is a modern Next.js application that allows users to explore countries around the world using real API data from the REST Countries API.

## Features

- App Router
- File-based routing
- Dynamic routes
- Server Components
- Client Components
- Real API data fetching
- Static rendering and caching
- Dynamic rendering
- Search functionality
- Region filtering
- Sort by population
- Dark mode
- Favorite countries
- Loading page
- Custom 404 page
- Responsive design
- Google Maps links

## API Used

REST Countries API

All countries:
https://restcountries.com/v3.1/all

Single country:
https://restcountries.com/v3.1/alpha/AFG

## Pages

- Home Page
- Countries Page
- Country Details Page
- Search Page
- About Page
